# Bible Stories

### Bible stories in Indian Sign language
<img src="/img/assets/biblestories1.png" width="1200px" alt="biblestories1" />

The stories are now available in Indian Sign Language. The user must choose **Indian sign Language** from the drop-down **(1)** menu in order to view the stories and select the stories **(2)**. 
The selected stories appear on the right pane  **(3)**. Click on the play button  **(4)** to watcth the videos. 


### Text version of stories

<img src="/img/assets/biblestories.png" width="1000px" alt="biblestories" />

The Landing page and the Study Bible page have access to **Bible Stories**.

The language drop-down menu **(1)** allows the user to select the desired language.

The user can now choose the desired stories from the index **(2)**.

The selected Bible Story can be seen in the right section **(3)**. Beautiful visuals have been used to illustrate the story.

The user has the option to adjust the font size by clicking on ‘three dots‘icon **(4)**.
