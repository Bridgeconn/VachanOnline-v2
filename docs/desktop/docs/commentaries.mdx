# Commentaries

To access the Commentaries the user must click on **Commentaries** icon **(1)**, select the Bible in the preferred language and version and choose a book and chapter from the drop-down menu **(2)**.

The preferred commentaries can be chosen from the Commentary drop-down menu **(3)**.

Some of the commentaries available are Mathew Henry Concise Commentary, ESV Global Study Bible and Bridgeway Bible Commentary.
Users can view the introduction page of the selected book by clicking the arrow at **(4)**.

<img src="/img/assets/commentary1.png" width="1000px" alt="commentary" />

If users have any questions about using the Commentaries, they can access the help page by clicking on **(5)**.
